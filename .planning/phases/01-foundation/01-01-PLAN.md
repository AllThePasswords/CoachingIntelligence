  <name>Task 1: Create Vite project with React and install dependencies</name>
  <files>
    package.json
    vite.config.js
    index.html
    src/main.jsx
    src/App.jsx
  </files>
  <action>
    1. Create new Vite project using React template in the project root:
       ```bash
       npm create vite@latest . -- --template react
       ```
       (Use `.` to create in current directory since we already have .planning/)

    2. Install React 19 (override default React 18):
       ```bash
       npm install react@^19.0.0 react-dom@^19.0.0
       ```

    3. Install Tailwind CSS 4 with Vite plugin:
       ```bash
       npm install tailwindcss@^4.0.0 @tailwindcss/vite
       ```

    4. Install Geist fonts (use Fontsource, NOT official geist package which requires Next.js):
       ```bash
       npm install @fontsource-variable/geist @fontsource-variable/geist-mono
       ```

    5. Update vite.config.js to add Tailwind plugin:
       ```javascript
       import { defineConfig } from 'vite';
       import react from '@vitejs/plugin-react';
       import tailwindcss from '@tailwindcss/vite';

       export default defineConfig({
         plugins: [
           react(),
           tailwindcss(),
         ],
       });
       ```

    6. Update src/main.jsx to import fonts:
       ```jsx
       import React from 'react';
       import ReactDOM from 'react-dom/client';
       import '@fontsource-variable/geist';
       import '@fontsource-variable/geist-mono';
       import './styles/index.css';
       import App from './App';

       ReactDOM.createRoot(document.getElementById('root')).render(
         <React.StrictMode>
           <App />
         </React.StrictMode>
       );
       ```

    7. Create minimal src/App.jsx that displays font test:
       ```jsx
       function App() {
         return (
           <div className="min-h-screen bg-bg-page p-8">
             <h1 className="text-3xl font-bold text-text-primary mb-4">
               Coaching Intelligence
             </h1>
             <p className="text-text-secondary mb-2">
               Geist Sans is working if this looks clean.
             </p>
             <code className="font-mono text-text-tertiary bg-bg-card px-2 py-1 rounded">
               CALL-1042 (Geist Mono)
             </code>
             <div className="mt-4 h-2 w-32 bg-card-gradient rounded"></div>
           </div>
         );
       }

       export default App;
       ```

    IMPORTANT: Do NOT create tailwind.config.js - Tailwind v4 uses CSS-first configuration via @theme directive.
  </action>
  <verify>
    Run `npm run dev` and confirm:
    - No console errors
    - Page loads at localhost:5173
    - Build succeeds: `npm run build`
  </verify>
  <done>
    Vite dev server starts, React 19 loads, Tailwind processes classes.
  </done>

<task type="auto">
  <name>Task 2: Configure design tokens in CSS with @theme directive</name>
  <files>
    src/styles/index.css
  </files>
  <action>
    Create src/styles/index.css with complete design system tokens:

    ```css
    @import "tailwindcss";

    @theme {
      /* Typography - Fonts */
      --font-sans: 'Geist Variable', -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
      --font-mono: 'Geist Mono Variable', ui-monospace, 'SF Mono', monospace;

      /* Colors - Backgrounds */
      --color-bg-page: #F5F5F5;
      --color-bg-card: #FFFFFF;

      /* Colors - Text (Neutral Gray Scale from design spec) */
      --color-text-primary: #171717;
      --color-text-secondary: #525252;
      --color-text-tertiary: #737373;
      --color-text-muted: #A3A3A3;

      /* Colors - Status */
      --color-status-success: #22C55E;
      --color-status-warning: #F59E0B;
      --color-status-error: #EF4444;
      --color-status-info: #3B82F6;

      /* Colors - Accent (Pink Gradient from design spec) */
      --color-accent-pink: #EC4899;
      --color-accent-purple: #A855F7;
      --color-accent-pink-light: #FDF2F8;

      /* Colors - Borders */
      --color-border-light: #E5E5E5;
      --color-border-default: #D4D4D4;
    }

    /* Base layer for global styles */
    @layer base {
      body {
        font-family: var(--font-sans);
        background-color: var(--color-bg-page);
        color: var(--color-text-primary);
      }
    }

    /* Custom utilities for card gradient */
    @layer utilities {
      .bg-card-gradient {
        background: linear-gradient(135deg, #FFFFFF 0%, #FDF2F8 100%);
      }

      .text-gradient-pink {
        background: linear-gradient(135deg, #EC4899 0%, #A855F7 100%);
        -webkit-background-clip: text;
        -webkit-text-fill-color: transparent;
        background-clip: text;
      }
    }
    ```

    This establishes:
    - Geist Sans as default body font
    - Geist Mono for code/citations
    - Neutral gray scale (NAV-05)
    - Pink gradient accent colors (NAV-05)
    - Page and card background colors
    - Status colors for badges
    - Custom gradient utilities

    All tokens become Tailwind utilities automatically:
    - `text-text-primary`, `text-text-secondary`, etc.
    - `bg-bg-page`, `bg-bg-card`
    - `font-mono` for Geist Mono
    - `bg-card-gradient` for pink gradient cards
